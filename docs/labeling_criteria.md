# 관상 특징 라벨링 기준 정의서

## 개요

MediaPipe Face Mesh 468개 랜드마크 좌표에서 **얼굴 대비 비율**을 계산하고,
이를 기반으로 관상 특징을 분류하는 3단계 파이프라인의 라벨링 기준.

```
1단계: 랜드마크 → 비율 수치 계산
2단계: 비율 수치 → 속성 분류 (대/중/소 등)
3단계: 속성 조합 → 관상 명칭 매핑
```

## 기준 랜드마크 (정규화 기준)

| 기준      | 랜드마크  | 설명                          |
| --------- | --------- | ----------------------------- |
| 얼굴 높이 | 10 → 152  | 이마 상단 ~ 턱 끝             |
| 얼굴 너비 | 234 → 454 | 좌측 관자놀이 ~ 우측 관자놀이 |
| 미간 중심 | 6         | 코 브릿지 상단                |

> 모든 비율은 `해당 부위 길이 / 기준 길이`로 계산. 사진 크기 무관.

## 1. 눈 (Eyes)

### 사용 랜드마크

- 우측 눈: 외측 33, 내측 133, 상안검 159/160/161, 하안검 145/153/154
- 좌측 눈: 외측 263, 내측 362, 상안검 386/387/388, 하안검 374/380/381
- 홍채: 468-477 (refine_landmarks=True)

### 측정 비율

| 속성                            | 계산 방법                         | 분류 기준                                                  |
| ------------------------------- | --------------------------------- | ---------------------------------------------------------- |
| **눈 크기** (eye_height_ratio)  | 눈 높이(159↔145 평균) / 얼굴 높이 | 소: <0.04, 중: 0.04~0.06, 대: >0.06                        |
| **눈 길이** (eye_width_ratio)   | 눈 너비(33↔133) / 얼굴 너비       | 단: <0.22, 중: 0.22~0.28, 장: >0.28                        |
| **눈 비율** (eye_aspect_ratio)  | 눈 높이 / 눈 너비                 | 세장형: <0.25, 보통: 0.25~0.40, 둥근형: >0.40              |
| **눈꼬리 각도** (eye_tilt_deg)  | 외측-내측 라인의 수평 대비 각도   | 처진눈: <-5, 수평: -5~5, 올라간눈: >5                      |
| **눈 간격** (eye_spacing_ratio) | 양 내측 거리(133↔362) / 얼굴 너비 | 좁음: <0.28, 보통: 0.28~0.35, 넓음: >0.35                  |
| **삼백안 여부** (sclera_ratio)  | 홍채 중심 y좌표의 눈 높이 내 위치 | 상삼백: 홍채 하단 편중, 하삼백: 홍채 상단 편중, 정상: 중앙 |
| **좌우 비대칭** (eye_asymmetry) | 좌/우 눈 면적 차이 비율           | 대칭: <10%, 비대칭: >=10%                                  |

### 관상 매핑 규칙

| 관상명        | 속성 조합 조건                                    |
| ------------- | ------------------------------------------------- |
| 봉안 (봉황안) | 세장형 + 장 + 올라간눈 + 대칭                     |
| 용안          | 대 + 장 + 올라간눈                                |
| 삼백안        | 삼백안 여부 = 상삼백 or 하삼백                    |
| 사백안        | 대 + 둥근형 + 삼백 심화 (홍채 면적/눈 면적 < 0.3) |
| 도화안        | 중~대 + 보통~둥근형 + 처진눈~수평                 |
| 원앙안        | 중 + 둥근형 + 수평 + 대칭                         |
| 음양안        | 비대칭 >= 10%                                     |
| 학안          | 세장형 + 장 + 수평                                |
| 우안          | 둥근형 + 처진눈 + 넓은 간격                       |
| 사안          | 세장형 + 소 + 올라간눈 (날카로운 인상)            |
| 취안          | 중 + 둥근형 + 처진눈 (취한 듯 느릿한 눈매)        |
| 호안          | 대 + 둥근형 + 올라간눈                            |
| 앵안          | 소~중 + 세장형 + 올라간눈 (앵무새 눈매)           |
| 공작안        | 대 + 장 + 수평 + 대칭                             |

## 2. 눈썹 (Eyebrows)

### 사용 랜드마크

- 좌측 눈썹: 70(내측), 63/105(정점), 46(외측)
- 우측 눈썹: 300(내측), 293/334(정점), 276(외측)

### 측정 비율

| 속성                                 | 계산 방법                     | 분류 기준                                     |
| ------------------------------------ | ----------------------------- | --------------------------------------------- |
| **눈썹 길이** (brow_length_ratio)    | 눈썹 너비(70↔46) / 얼굴 너비  | 단: <0.18, 중: 0.18~0.25, 장: >0.25           |
| **눈썹 두께** (brow_thickness_ratio) | 눈썹 상하 폭 / 얼굴 높이      | 가늘: <0.015, 보통: 0.015~0.025, 굵은: >0.025 |
| **눈썹 각도** (brow_tilt_deg)        | 내측→외측 기울기 각도         | 하강형: <-5, 수평: -5~5, 상승형: >5           |
| **눈썹 곡률** (brow_curvature)       | 정점 높이 / 눈썹 너비         | 직선: <0.05, 완만: 0.05~0.12, 아치형: >0.12   |
| **눈썹 간격** (brow_gap_ratio)       | 양 내측 간 거리 / 얼굴 너비   | 좁음: <0.08, 보통: 0.08~0.14, 넓음: >0.14     |
| **눈-눈썹 간격** (brow_eye_gap)      | 눈썹 하단~눈 상단 / 얼굴 높이 | 좁음: <0.03, 보통: 0.03~0.06, 넓음: >0.06     |

### 관상 매핑 규칙

| 관상명              | 속성 조합 조건                          |
| ------------------- | --------------------------------------- |
| 일자눈썹            | 수평 + 직선                             |
| 초승달눈썹          | 완만~아치형 + 가늘~보통                 |
| 유엽미 (버들잎눈썹) | 장 + 가늘 + 완만                        |
| 검미 (칼눈썹)       | 굵은 + 직선 + 상승형                    |
| 팔자눈썹            | 하강형 + 완만                           |
| 삼각눈썹            | 상승형 + 직선 + 굵은                    |
| 단미                | 단 + 굵은                               |
| 산미 (흩어진눈썹)   | 눈썹 밀도 불균일 (랜드마크 간격 불규칙) |
| 농미 (짙은눈썹)     | 굵은 + 장                               |
| 담미 (옅은눈썹)     | 가늘 + 단                               |
| 연미 (이어진눈썹)   | 눈썹 간격 매우 좁음 (<0.04)             |

- 콧등: 168

### 측정 비율

| 속성                              | 계산 방법                          | 분류 기준                                    |
| --------------------------------- | ---------------------------------- | -------------------------------------------- |
| **코 길이** (nose_length_ratio)   | 코 길이(6↔1) / 얼굴 높이           | 단: <0.25, 중: 0.25~0.33, 장: >0.33          |
| **코 너비** (nose_width_ratio)    | 콧볼 너비(48↔278) / 얼굴 너비      | 좁음: <0.22, 보통: 0.22~0.30, 넓음: >0.30    |

| **코 비율** (nose_aspect)         | 코 길이 / 콧볼 너비                | 넓적: <0.9, 균형: 0.9~1.2, 길쭉: >1.2        |

| 매부리코      | 높음 + 장 + 매부리 각도         |
| 들창코        | 들창코 각도 + 단~중             |
| 단코          | 단 + 보통~좁음                  |
| 주먹코        | 넓음 + 코끝 둥근 + 낮음~보통    |
| 독수리코      | 높음 + 장 + 매부리 + 좁음       |
| 사자코        | 넓음 + 높음 + 장                |

## 7. 귀 (Ears)

### MediaPipe 한계

> MediaPipe Face Mesh는 귀를 상세하게 감지하지 못합니다.
> 귀 부착 지점만 대략적으로 감지 가능 (172, 173 근처).

### 대안 전략

1. **별도 감지 모델** 사용 (dlib 또는 커스텀 귀 감지 모델)
2. **사용자 입력** 보조 (귀 사진 별도 업로드 또는 선택지 제공)
3. **Phase 2로 이관** — MVP에서는 귀 분석 제외

### 측정 속성 (별도 모델 사용 시)

| 속성             | 분류 기준                        |
| ---------------- | -------------------------------- |
| **귀 크기**      | 소, 중, 대 (귀 높이 / 얼굴 높이) |
| **귓불 크기**    | 소, 중, 대 (귓불 / 귀 전체 비율) |
| **귀 부착 형태** | 부착형, 분리형                   |
| **귀 돌출**      | 밀착, 보통, 돌출                 |

| 속성                                  | 계산 방법                        | 분류 기준                               |
| ------------------------------------- | -------------------------------- | --------------------------------------- |
| **인중 길이** (philtrum_length_ratio) | 코 하단(2)↔윗입술(0) / 얼굴 높이 | 짧음: <0.05, 보통: 0.05~0.08, 긴: >0.08 |
| **인중 너비** (philtrum_width_ratio)  | 인중 양측 거리 / 코 너비         | 좁음: <0.3, 보통: 0.3~0.5, 넓음: >0.5   |
| **인중 깊이** (philtrum_depth)        | z좌표 기반 홈 깊이               | 얕음, 보통, 깊음                        |

### 관상 매핑 규칙

| 관상명      | 속성 조합 조건   |
| ----------- | ---------------- |
| 긴 인중     | 긴 + 보통~넓음   |
| 짧은 인중   | 짧음             |
| 뚜렷한 인중 | 보통~넓음 + 깊음 |
| 평평한 인중 | 얕음 + 넓음      |
| 좁은 인중   | 좁음 + 깊음      |
| 넓은 인중   | 넓음 + 얕음      |

## 10. 광대뼈 (Cheekbones)

### 사용 랜드마크

- 광대 돌출: 234(좌), 454(우) 근처 중안부 랜드마크
- 중안부 측면: 93(좌), 323(우)

### 측정 비율

| 속성                                   | 계산 방법                     | 분류 기준                                 |
| -------------------------------------- | ----------------------------- | ----------------------------------------- |
| **광대 너비** (cheekbone_width_ratio)  | 광대 양측 거리 / 얼굴 너비    | 좁음: <0.90, 보통: 0.90~0.98, 넓음: >0.98 |
| **광대 돌출도** (cheekbone_protrusion) | 광대 z좌표 vs 주변 z좌표 편차 | 평탄: 편차 낮음, 보통, 돌출: 편차 높음    |
| **광대 위치** (cheekbone_position)     | 광대 최대폭 y좌표 / 얼굴 높이 | 높음: <0.45, 보통: 0.45~0.55, 낮음: >0.55 |

### 관상 매핑 규칙

| 관상명      | 속성 조합 조건   |
| ----------- | ---------------- |
| 높은 광대   | 높음 + 돌출      |
| 넓은 광대   | 넓음 + 돌출      |
| 평평한 광대 | 평탄 + 보통 위치 |
| 낮은 광대   | 낮음 + 보통~평탄 |
